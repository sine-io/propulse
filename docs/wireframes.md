# 页面线框：房脉 / propulse

## 1. 首页线框

### 页面结构

```text
[顶部导航]
Logo | 换房测算 | 目标小区 | 出手窗口 | 判断方法 | 工具模板 | 登录

[首屏 Hero]
标题：想买房或换房，先算清压力，再判断时机
副标题：输入你的预算和目标小区，判断现在能不能买、压力有多大、是否适合看房、等待、砍价或出手
[主按钮：开始换房测算]
[次按钮：添加目标小区]
右侧：换房压力卡 + 小区信号卡 + 行动建议卡

[核心模块 1：换房能力]
安全总价 | 勉强总价 | 危险总价 | 月供压力 | 资金缺口

[核心模块 2：目标小区观察]
挂牌变化 | 降价房源 | 成交区间 | 供应压力 | 流动性

[核心模块 3：出手窗口判断]
看 | 等 | 砍价 | 出手

[核心模块 4：边用边学]
为什么不能只看挂牌价 / 什么是买方窗口 / 改善为什么要看新旧房价差

[工具区]
换房预算表 | 小区观察表 | 周监测表 | 看房记录表 | 谈价清单
```

### 视觉说明

- 白底，深蓝 + 青绿色为主
- 首屏要像决策工具，不像课程首页
- 卡片表达“结论 + 依据 + 下一步”
- 信息密度高，但避免专家报告感

## 2. 换房测算页线框

### 页面结构

```text
[页面标题]
先算清楚：你现在能不能换，换完压力有多大

[左侧输入区]
当前现金
旧房估值
旧房贷款余额
家庭月收入
当前月供
可接受月供
目标房源总价
装修预算
税费 / 中介费
过渡租房成本

[右侧结果区]
安全总价：xxx 万
勉强总价：xxx 万
危险总价：xxx 万
月供压力：安全 / 偏高 / 危险
首付缺口：xxx 万
旧房最低安全成交价：xxx 万

[策略建议]
建议：先卖后买 / 同步推进 / 暂缓改善
原因：现金流、旧房回款、目标总价、月供压力

[方法解释]
为什么月供安全线比总价更重要？
为什么改善换房要留过渡资金？
```

### 视觉说明

- 左侧像计算器，右侧像体检报告
- 结果要清楚分层：安全、勉强、危险
- 每个风险提示都要有解释，不要只给红色警告

## 3. 目标小区详情页线框

### 页面结构

```text
[顶部]
小区名称 | 板块 | 加入观察池 | 设置提醒

[当前判断卡]
当前状态：继续观察 / 适合砍价 / 可以重点看 / 价格偏硬 / 暂不建议追
一句话结论：挂牌增加、降价变多，但成交偏弱，房东预期开始松动
下一步：可以开始看房，但不急下定，重点砍总价 xxx 万以内房源

[价格与供应]
当前挂牌价格带
成交价格区间
挂牌成交差
在售套数变化
新增挂牌
降价房源

[趋势区]
挂牌均价趋势
成交区间趋势
在售量趋势
降价频率趋势
成交周期趋势

[目标户型]
两房 / 三房 / 改善户型价格带
同户型近期挂牌与降价情况

[判断依据]
为什么当前判断为“适合砍价”
哪些信号支持
哪些风险需要继续观察

[操作]
加入观察池 | 设置降价提醒 | 添加看房记录 | 导出谈价清单
```

### 视觉说明

- 顶部先给结论，再给数据
- 数据服务于行动，不堆图表
- 每个指标旁边都能展开“这说明什么”

## 4. 出手窗口页线框

### 页面结构

```text
[页面标题]
现在适合看、等、砍价，还是出手？

[综合判断]
行动建议：看 / 等 / 砍价 / 出手
信心等级：低 / 中 / 高

[判断矩阵]
预算是否安全
旧房回款是否可控
目标小区是否松动
降价房源是否增加
成交是否支撑价格
目标户型是否稀缺
替代小区是否更优

[行动清单]
本周应该做什么
哪些房源可以约看
哪些价格可以尝试砍
什么情况下应该暂停

[风险提示]
现金流风险
旧房卖不出风险
目标小区价格仍偏硬
房源稀缺导致被动追价
```

### 视觉说明

- 像“决策仪表盘”
- 不只展示行情，还要合并用户预算
- 重点是下一步动作

## 5. 判断方法页线框

### 页面结构

```text
[标题]
不是记术语，而是学会判断

[问题式目录]
为什么不能只看挂牌价？
什么是买方窗口？
挂牌变多但成交弱，说明什么？
降价房源变多，一定会继续跌吗？
改善换房为什么要看新旧房价差？
为什么月供安全线比总价重要？
如何做每周小区复盘？

[单篇结构]
真实问题
错误判断
正确看法
关键指标
例子
你应该怎么用

[练习区]
选择一个目标小区
记录本周挂牌、降价、成交变化
写下本周行动：看 / 等 / 砍价 / 出手
```

### 视觉说明

- 不做传统课程目录
- 以真实问题组织内容
- 每篇内容都要回到用户下一步行动
