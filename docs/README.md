## 当前产品方向

房脉已经从“房产监测与学习平台”收敛为：

> 面向刚需与改善用户的换房决策助手。

核心不是泛泛地解释房价，而是回答：

- 我现在能不能买？
- 我能买多少钱？
- 换房压力有多大？
- 目标小区现在贵不贵？
- 现在适合看、等、砍价，还是出手？
- 我如何在这个过程中建立自己的判断方法？
